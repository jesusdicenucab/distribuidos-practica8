## Description

Se requiere que su **Grupo** diseñe e implemente un Servicio Web de tipo REST que cumpla con los siguientes requerimientos:

* Listar, crear, actualizar, eliminar Facultades.
* Listar, crear, actualizar, eliminar Escuelas.
* Listar, crear, actualizar, eliminar Secciones.
* Listar, crear, actualizar, eliminar Personas.
* Inscribir y eliminar la inscripción de una persona a una sección.
* Listar los estudiantes de una sección
* Listar los profesores de una sección

Consideren los siguiente puntos:

* Al crear un objeto se debe agregar la fecha de creación respectiva.
* La eliminación es lógica. Se coloca como estado “disable” y se agrega la respectiva fecha.
* Los atributos de fecha de creación y eliminación son inmutables desde el servicio web.
* Nunca se listan objetos eliminados.
* Las fechas deben mostrarse en formato ISO 8601, Ejemplo 2020-07-26T01:00:00+04:00.
* Para la resistencia de datos, podrán usar archivos o un servicio de base de datos.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
